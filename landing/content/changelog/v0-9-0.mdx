## Graph Explorer

The new Graph Explorer provides a visual interface for exploring your agent's memory graph. Features include:

- **Force-directed layout** — Interactive graph visualization with zoom, pan, and drag.
- **Node filtering** — Filter by entity type, relationship, or time range.
- **Path traversal** — Find connections between any two entities in the graph.

## Community Detection

Added community detection algorithms that automatically identify clusters of related entities in your agent's memory. This enables:

- **Topic clustering** — Group related conversations and facts.
- **Anomaly detection** — Identify isolated or unusual memory patterns.
- **Memory pruning** — Archive or compress low-value memory clusters.

## Performance Improvements

- Reduced memory retrieval latency by 40% for large graphs (>100k nodes).
- Improved batch ingestion throughput by 3x.
- Optimized query planning for complex graph traversals.
