## What's New

After months of development, we're thrilled to announce the public beta of OpenZync — the first persistent, graph-based memory infrastructure purpose-built for AI agents.

### Core Features

- **Multi-Graph Backends** — Choose from Neo4j, FalkorDB, Memgraph, and more. Switch between backends without changing your agent code.
- **Graph-Based Memory** — Store agent interactions as structured knowledge graphs with entities, relationships, and facts.
- **Multi-LLM Support** — Route tasks to OpenAI, Anthropic, Google, or local models based on cost, latency, or capability requirements.
- **Human-in-the-Loop** — Define approval gates for any side-effecting action. Every escalation is logged and auditable.

### Tool Ecosystem

- **Plugin System** — Extend agents with a typed, idempotent plugin interface.
- **Deterministic Workflows** — Stage-gated DAGs for production reliability, with crash recovery and resume capability.
- **Built-in Retry** — Exponential backoff with circuit breakers on every external call.

### Observability

- **Structured Logging** — Every tool execution, LLM call, and workflow transition emits structured logs with latency, token counts, and cost estimates.
- **Credit Billing** — Usage-based billing with configurable limits and rate limiting.
